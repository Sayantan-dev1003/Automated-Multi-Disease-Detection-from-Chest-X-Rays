Sample dataset generated by create_sample_dataset.py

Contents:
- images/                 -> 20 copied sample image files
- sample_manifest.csv     -> manifest with paths relative to this folder

How to use:
- copy the 'images' folder into your repo's data/sample/images/
- place sample_manifest.csv in data/sample/
- manifest.csv paths are relative (e.g. images/img001.png)

Note: This is a tiny subset for local testing. Do NOT commit full dataset to git.
